
# Medicare Claims Analysis

## File Information
- **Program Name:** `medicare_claims_quantitative_analysis.ipynb`
- **Author:** Stephen Lew
- **Last Updated:** 4/13/2026

## Description
Built a cloud-integrated Python/SQL framework to analyze CMS Medicare claims. Designed a reusable ETL pipeline via Jinja2 and Snowflake, conducted multivariable regression to predict spending trends, and delivered automated financial reports with Pandas and Seaborn.
